# 3D 手势交互粒子系统

Three.js + Canvas 肤色检测，通过摄像头识别手指数量控制粒子群形成不同文字。

## 功能

- **手势 1（食指）**：粒子组成 "hello"
- **手势 2（食指+中指）**：粒子组成 "哎呀不错哦"
- **手势 3（食指+中指+无名指）**：粒子组成 "我爱你们"
- **手掌张合**：控制粒子群收缩与扩散
- **手指靠近粒子**：产生引力吸引效果
- **无手势**：粒子散布为球体

## 技术栈

- Three.js (CDN) — 3D 渲染
- Canvas API — 肤色检测 + 手指计数（零依赖）
- WebRTC (getUserMedia) — 摄像头采集

## 使用方式

1. 浏览器打开 `index.html`
2. 允许摄像头权限
3. 手心面向摄像头，伸出手指即可交互
4. 键盘 `1` `2` `3` `0` 也可切换文字形态

> 需在 HTTP(S) 协议下打开（不能是 `file://`），摄像头才能工作。可以用 VS Code Live Server 或 `npx serve .` 启动本地服务器。

## 文件

- `index.html` — 完整应用（HTML + CSS + JS）
- `server.js` — 本地开发服务器（Node.js）
